# Uppgift1-StenSaxP-se
Använder TKinter för att skapa ett sten sax påse-spel

Spelet är gjort i Python med hjälp av TKiner och spelas mellan en spelare och datorn.

1, Spelaren gör ett val
spelaren klickar på en av tre knapper:
Sten, Sax eller Påse.
När man klickar på en kanpp sparas spelarens val.

2. Datorn gör ett slumpmässigt val
Efter spelaren har valt, väljer datorn slumpmässigt mellan:
Sten, Sax eller Påse.

Spelet avgör vem som vinner
Både spelarens och datorns val omvadlas till siffror:
Sten = 0
Sax = 1
Påse = 2
Med hjälp av dessa siffrorr räknar programmet ut resultatet:
Om båda väljer samma = oavgjort
Om spelarens val slår datorns = spelaren vinner
Annars vinner datorn

Poängen uppdaterasOm spelaren vinner ökar spelarens poäng med 1
Om datorn vinner ökar datorns poäng med 1
Vid lika ändras inga poäng

Resultatet visas i fönstret
Iprogramfönstret visas:
Vad spelaren valde
Vad datorn valde
Vem som vann omgången
Spelarens poäng
Datorns poäng

Detta uppdateras varje gång spelaren klickar på en knapp
